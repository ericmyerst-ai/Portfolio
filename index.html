System Blueprint: Sane Club Automated Curation Engine

How do you streamline enterprise procurement and eliminate data fatigue without risking AI inaccuracies? This functional prototype demonstrates a high-integrity procurement engine designed to filter massive hardware inventories down to three optimal options based on rigid business constraints.

  • Deterministic Business Logic: Designed a constraint-matching architecture that evaluates user operational profiles against structural and spatial realities, ensuring 100% reliable alignment without the risk of LLM hallucinations.
  • Hard Constraint Filtering: Engineered a systemic boundary mechanism that automatically disqualifies hardware assets failing to meet baseline power, compliance, or footprint requirements.
  • Lifecycle Data Governance: Integrated a "Longevity Profile" framework into the core data registry to prioritize repairability indexes, asset depreciation data, and sustainable life cycles.
  • Accelerated System Prototyping: Utilized advanced natural language orchestration and AI-assisted workflow engines to rapidly model, test, and deploy a fully functional front-to-back system architecture.

FocusPivot: Digital Detox Interceptor

An Android utility application built using Kotlin and Jetpack Compose designed to break unconscious phone scrolling habits. When a user launches a target app, the system interrupts the behavior with a mindful pattern-interrupt and offers a productive pivot to remote career opportunities.

  • Seamless Interception: Runs an active Foreground Service utilizing system-level OS APIs to scan running packages in real-time.
  • Mindful Interruption: Blocks target apps (like Google Chrome, Instagram, or TikTok) with a 5-second animated breathing loop to disrupt automatic muscle memory.
  • Configurable Settings: Uses on-device SharedPreferences to allow users to toggle their blocklist settings directly inside the UI.
  • Temporary Bypass: Implements a 2-minute unblock window, allowing users to safely return to their task before the lock re-engages.
  • Live Career Feed: Queries the Jobicy remote job board API dynamically in real-time to load active English-speaking career listings.

These projects were rapid-prototyped using AI-assisted development ("vibe coding"),
leveraging LLMs to iterate on functional system logic and interface design.
