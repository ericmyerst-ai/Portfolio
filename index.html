Sane Club: Wholesale Curation Engine

How fast can you go from an idea to a functional procurement engine? In this demo, I showcase "Sane Club," an intelligent hardware curation platform built to eliminate choice paralysis.

  • Intelligent Procurement: Uses a backend matching engine to balance user-defined rituals, spatial constraints, and financial values.
  • Hard Constraint Logic: The system automatically culls hardware that violates specific power or footprint requirements.
  • Longevity-First: Features a "Longevity Profile" prioritizing high repairability to combat planned obsolescence.
  • Rapid Prototyping: Built using AI-assisted development ("vibe coding").
