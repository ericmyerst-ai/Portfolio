# Sane Club: Wholesale Curation Engine

How fast can you go from an idea to a functional procurement engine?

In this demo, I showcase "Sane Club," an intelligent hardware curation platform built to eliminate choice paralysis. Unlike standard search tools, Sane Club uses a custom "Match Weight Diagnostic" to filter millions of items down to three expertly-vetted, high-integrity nominees.

## Project Highlights

* **Intelligent Procurement**: Uses a backend matching engine to balance user-defined rituals, spatial constraints, and financial values.
* **Hard Constraint Logic**: The system automatically culls hardware that violates specific power or footprint requirements, ensuring 100% viability for the user.
* **Longevity-First**: Every item features a "Longevity Profile," prioritizing high repairability and disassembly ratios to combat planned obsolescence.
* **Rapid Prototyping**: This project was built using AI-assisted development ("vibe coding"), focusing on high-velocity iteration of complex matching logic and dynamic interface grids.
