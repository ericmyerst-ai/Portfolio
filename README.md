## 🛠️ Featured Projects

### 1. FocusPivot: Digital Detox Interceptor (Android / Kotlin)
FocusPivot is a native Android utility application designed to break unconscious phone scrolling habits. When a user attempts to open pre-selected distracting apps (such as Google Chrome, Instagram, or TikTok), a background service intercepts the launch, triggers a brief mindful breathing pattern-interrupt, and offers a productive pivot to active, English-speaking remote career opportunities.

*   **Mindful Interception:** Employs a visual 5-second breathing countdown to break automatic muscle-memory loops.
*   **System-Level Persistence:** Implements a native Android Foreground Service using modern API guidelines (API 34+ compatibilities).
*   **SharedPreferences Config:** Allows users to toggle which apps to block directly from the UI.
*   **Temporary Bypass:** Incorporates a 2-minute temporary bypass window saved to an in-memory `companion object` to eliminate disk I/O latency.
*   **Live Career Pivot:** Asynchronously fetches real-world, active remote jobs in English from the public Jobicy developer API.
*   📂 **Codebase & Implementation:** [FocusPivot GitHub Repository](https://github.com/ericmyerst-ai/DigitalDetox) *(or add your link)*

### 2. Sane Club: Wholesale Curation Engine (Web Procurement)
Sane Club is an intelligent hardware curation platform built to eliminate choice paralysis and simplify corporate hardware procurement. 

*   **Intelligent Procurement:** Uses a backend matching engine to balance user-defined rituals, spatial constraints, and financial values.
*   **Hard Constraint Logic:** The system automatically culls hardware that violates specific power or footprint requirements.
*   **Longevity-First Index:** Features a "Repairability Score" prioritizing high repairability to combat planned obsolescence.
*   📂 **Executive Briefing:** Check out the [Sane Club Executive Briefing PDF](Sane%20Club%20Executive%20Briefing.pdf) located in this repository.
